# MedInsure-AI-Medical-Insurance-Cost-Predictor

A Full Stack Machine Learning Web Application that ensures the data and predicts the estimated cost of Medical insurance based on user health and lifestyle data using Machine Learning - Supervised Learning Model (Linear Regression) built using Flask - Backend Services.

# 🚀 Project Overview

MedInsure Allows users to :

  💰 Predict medical insurance costs
  
  ⚠️ Analyze health risk levels
  
  📊 View analytics dashboard
  
  💾 Store and track prediction history

This Project Demonstrates the End-to-End Machine Learning Deployment with Pipeline.

This Contains Data Science + Full Stack Integration + PostgreSQL Integration.
